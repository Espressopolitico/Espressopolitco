# Espressopolitco
Espresso Politico è uno spazio online che spiega la politica italiana ed estera in modo semplice e diretto. Nasce per aiutare i giovani a capire cosa succede nelle istituzioni e nel mondo, senza linguaggio complicato o toni di parte. Informazioni chiare, rapide e affidabili per orientarsi meglio nell’attualità.
